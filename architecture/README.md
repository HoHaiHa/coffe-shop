# Kiến trúc Hệ thống (Architecture)

Thư mục này chứa các tài liệu liên quan đến thiết kế kiến trúc của hệ thống Coffee Shop, bao gồm:

- Kiến trúc tổng thể (High-level architecture).
- Thiết kế cơ sở dữ liệu (ERD diagrams).
- Kiến trúc triển khai (Deployment diagram).
- Các quyết định về công nghệ và design pattern (Architecture Decision Records - ADRs).

> **Lưu ý:** Việc thay đổi thiết kế cơ sở dữ liệu hoặc luồng kiến trúc cần phải được review và cập nhật tại đây trước khi áp dụng vào code.
